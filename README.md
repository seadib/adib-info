# 🧑‍💻 Abdullah Al Adib — Personal Portfolio Website

A modern, animated, and fully functional personal portfolio website. Visitors can view my profile, projects, skills, journey timeline, and send messages directly to my email via the contact form.

## 🌐 Live Demos

- **GitHub Pages:** [https://seadib.github.io/adib-info/](https://seadib.github.io/adib-info/)
- **Netlify:** [https://adib-info.netlify.app/](https://adib-info.netlify.app/)

---

## 📌 Features

- ✅ **Dark / Light Theme Toggle** – Switch themes based on user preference
- ✅ **Particle Animation** – Interactive background particles
- ✅ **Cursor Glow Effect** – A gradient glow that follows the mouse
- ✅ **Scroll Progress Bar** – Shows how much of the page has been viewed
- ✅ **Staggered Animations** – Elements appear step by step on scroll
- ✅ **Timeline** – Visual representation of the journey
- ✅ **Project Card Modal** – Click to confirm before opening projects in a new tab
- ✅ **Web3Forms Integration** – Send form data directly to email without a backend
- ✅ **Visitor Counter** – Tracks visit count using localStorage
- ✅ **Responsive Design** – Adapts to mobile, tablet, and desktop screens

---

## 🛠️ Technologies Used

| Technology | Purpose |
|------------|---------|
| **HTML5** | Page structure |
| **CSS3** | Styling, animations, theme variables |
| **JavaScript (Vanilla)** | Interactions, animation control, API calls |
| **Google Fonts** | `Plus Jakarta Sans` (English) & `Noto Sans Bengali` (Bengali) fonts |
| **Web3Forms API** | Send contact form data to email |
| **localStorage** | Save theme preference and visitor count |
| **Canvas API** | Particle animation |
| **Intersection Observer API** | Trigger scroll-based animations |
| **GitHub Pages / Netlify** | Hosting platforms |
