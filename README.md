# Abdullah Al Adib — Personal Portfolio

A modern, glassmorphic portfolio website built for **Abdullah Al Adib**, student at Dhaka College (HSC 2027) & aspiring software engineer.  
Features Apple‑inspired liquid glass UI, dark/light mode, animated particles, and contact links.

## 🔥 Features
- Glassmorphism + backdrop blur
- Dark / Light mode toggle
- Liquid blob animation
- Particle network canvas
- Responsive design
- Project cards with modal
- Contact form (demo)
- Social links (Email, Facebook, Messenger, WhatsApp, GitHub)

## 🛠 Tech Stack
- HTML5, CSS3, Vanilla JavaScript
- Canvas API, Google Fonts
